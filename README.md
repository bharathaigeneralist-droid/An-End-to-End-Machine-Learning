# An-End-to-End-Machine-Learning
Built an end-to-end HR analytics solution to predict employee attrition using machine learning techniques. Performed data preprocessing, feature engineering, and model evaluation to identify key factors driving employee turnover. Delivered actionable insights to help organizations improve retention strategies and workforce planning.

## Key Features of Your Project 
These highlight what your project does:

✔ End-to-End ML Pipeline
Data ingestion from Excel
Data cleaning (nulls, duplicates, constant columns)
Feature engineering (e.g., tenure-based ratios)
Model training, evaluation, and comparison

✔ Advanced Data Preprocessing
Removed low-variance features (EmployeeCount, Over18, StandardHours)
Handled missing values
Encoded categorical variables
Target transformation (Attrition → 0/1)

✔ Class Imbalance Handling
Applied SMOTE (Synthetic Minority Oversampling Technique)
Improved minority class prediction (attrition cases)

✔ Multiple ML Models
Logistic Regression
Decision Tree
Random Forest
XGBoost
PCA (Dimensionality Reduction)

✔ Model Evaluation Framework
Accuracy
Precision
Recall
F1 Score
Comparison across models

✔ Data Visualization
Attrition distribution plots
EDA insights using Matplotlib & Seaborn
